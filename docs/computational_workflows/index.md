# Computational Workflows

Document analyses so another lab member can rerun them without reconstructing assumptions from notebooks, terminal history, or Slack threads.

## Pages

- [RNA-seq Differential Expression](rna_seq_differential_expression.md)
- [Variant QC Summary](variant_qc_summary.md)

## Minimum Information

Each workflow should include:

- Purpose and scientific question.
- Inputs and where they are stored.
- Software versions or environment file.
- Commands or notebook order.
- Outputs and how to interpret them.
- Known limitations.
- Person responsible for review.
