# Data Management

This section records where lab data live, how files are named, and how project handoffs should work.

## Pages

- [File Naming and Storage](file_naming_and_storage.md)

## Core Expectations

- Raw data are read-only after receipt.
- Derived data include enough metadata to trace how they were produced.
- Shared locations are documented before a project changes hands.
- Sensitive data stay only in approved storage locations.
